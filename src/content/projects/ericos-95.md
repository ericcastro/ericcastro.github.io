EricOS 95 is the project you are currently looking at: a personal website that treats content like little desktop applications instead of plain web pages.

The goal was not just to make a nostalgic theme, but to build something that feels like an environment: draggable windows, fake applications, old-web jokes, and enough polish to make the whole thing feel intentional.

- Built as a static Astro site for GitHub Pages
- Uses Markdown for posts and project content
- Recreates a Windows 95 desktop and browser metaphor on top of a modern stack
- Includes dark mode, fake dialogs, start menu behavior, and all the unnecessary details I enjoy too much

It is probably the clearest summary of my current brain state: retro computing nostalgia colliding with modern tooling and a willingness to overbuild personal projects for fun.
