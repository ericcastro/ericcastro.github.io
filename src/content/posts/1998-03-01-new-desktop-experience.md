This website is now fully contained inside a fake Microsof Internet Exploder 3 window, because ordinary layouts are for the weak. Instead of navigating away, the important sections open as separate Windows 95 widgets floating on the desktop.

This means the browser is the main content container, but it still behaves like one more program on the desktop. Very important innovation.
